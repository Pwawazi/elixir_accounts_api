# Accounts and Users API

Environment variables are sourced from the .env file. 
The environment variables needed are in the .env.example file. Fill it with your application specific values.
Source the file manually or through a systemd/service application.

To start your Phoenix server:

  * Run `mix setup` to install and setup dependencies
  * Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:<YOUR_SERVING_PORT>`](http://localhost:<YOUR_SERVING_PORT>) from your browser.

Ready to run in production? Please [check our deployment guides](https://hexdocs.pm/phoenix/deployment.html).
